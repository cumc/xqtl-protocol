Genotype data partition by region#

This step is necessary for finemapping using SuSiE. TADB_enhanced_cis.bed is a list of TADB extended region for each gene, in formatting of gene_id, chr, start, end. The complete version of the file has been uploaded t o GitHub. Here we use a trimmed version that only contains information for the data we are analyzing in the MWE, ie, the protocol_example.protein.enhanced_cis_chr21_chr22.bed

sos run pipeline/genotype_formatting.ipynb genotype_by_region \
    --region-list protocol_example/protocol_example.protein.enhanced_cis_chr21_chr22.bed \
    --genoFile protocol_example/protocol_example.genotype.chr21_22.bed \
    --cwd output \
    --container containers/bioinfo.sif
